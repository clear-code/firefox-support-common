ESR60からESR68の変更点
======================

変更ログ
--------

### Firefox 61

 * ロケーションバー内の「...」（ページアクションメニュー）からOpenSearch Pluginによる検索エンジンを登録できるようになった。
 * WindowsとmacOSにおいて、ロケーションバー内の「...」（ページアクションメニュー）から「共有」でOS標準のリンク共有機能を呼び出せるようになった。
 * TLS1.3への対応が改善された。
 * http:// https:// のページから ftp:// のURLへのリンクは読み込みがブロックされるようになった。
 * 組み込みのテーマとして「ダーク」が追加された。
 * 拡張機能でタブを隠せるようになった。
 * ブックマークの同期処理が改善された。
 * 設定画面の構成を変更。「ホーム」カテゴリから起動時や新規タブの内容を変更でき、新規タブページからもこの設定画面に遷移できるようになった。
 * セキュリティ対策ソフトのAvastとAVGでHTTPSスキャン時にSSL_ERROR_RX_RECORD_TOO_LONGのエラーが表示される。（セキュリティ対策ソフトの更新が必要）

Links

 * https://www.mozilla.org/en-US/firefox/61.0/releasenotes/
 * https://bugzilla.mozilla.org/buglist.cgi?f1=cf_tracking_firefox_relnote&o1=equals&v1=61%2B

### Firefox 62

 * ホーム画面のサムネイルの4行表示に対応。
 * タブのコンテキストメニューから「コンテナーを開き直す」でタブを開き直せるようになった。
 * シマンテックの証明書がすべて無効化された。 security.pki.distrust_ca_policy = 0 と設定すると従来の挙動に戻せる。
 * CSSシェイプに対応。
 * CSS Variable Fontに対応。
 * autoconfig.cfgがサンドボックス内で実行されるようになった。general.config.sandbox_enabled = false で従来の挙動に戻せる。
 * ブックマークの「説明」フィールドを削除。
 * macOS 10.14で、OS自体がダークモードの時はFirefoxのテーマもダークになるようにした。
 * security.pki.name_matching_modeの初期値が3に変更され、証明書の検証がより厳密になった。
 * macOSで、Adobe Flashがより安全なサンドボックス内で実行されるようになった。
 * WebRTCでの画面共有時、画面を共有するウィンドウは最前面になるようになった。

### Firefox 63

 * Windows 10のOS自体の配色設定（「規定のアプリ モード」の「白」「黒」）に配色を合わせるようになった。
 * コンテンツブロッキングの機能が改善され、トラッキングCookieの制御などを統一されたUIで行えるようになった。
 * 複数のウィンドウやタブを開いた状態での終了時に、次回起動時にセッションを復元するかどうかの訊ね方が変わった。
 * OSのアクセシビリティ設定に従ってアニメーション効果を抑制するようになった。
 * ブックマークされたページをサイドバーで開く機能を削除。
 * 設定画面から自動更新を無効化できなくなった。（Policy EngineでDisableAppUpdateを設定することで代貸）
 * Ctrl-Tabの既定の挙動が、タブの並び順ではなく最終アクセス順での切り替えに変わった。（設定変更で元の挙動に戻せる）
 * Web Componentsのカスタム要素とシャドーDOMに対応。
 * セキュリティ対策ソフトのQuick HealがWindows用32bit版のFirefoxでクラッシュする。

### Firefox 64

 * タブの複数選択操作に対応。
 * about:performance でタブごと・アドオンごとのリソース消費状況を確認できるようになった。
 * アドオンのツールバーボタンの右クリックメニューからアドオンをアンインストールできるようになった。
 * macOSでconfiguration profilesを使ってポリシー設定を制御できるようになった。
 * ライブブックマークのプレビュー表示に対応しなくなった。
 * Symantecの証明書が完全に無効化された。
 * macOSで入力中の内容の前後に「www.」と「.com」を付与するキーボードショートカットをCtrl-EnterからCommand-Enterに変更した。

### Firefox 65

 * トラッキング保護のUIが整理された。
 * 複数の言語パックを切り替えられるようになった。
 * macOSのHandoff機能に対応。閲覧中のタブを別のiOSデバイスで開けるようになった。
 * セッション復元機能を有効にしている場合でも、Firefoxを終了して良いかどうか訊ねるようになった。
 * about:performance でアドオンやタブごとのメモリ使用状況を確認できるようになった。
 * Webページのスクリプトから同時に複数のポップアップが開かれる場面について、2つ目以降を自動的にブロックするようになった。
 * 32bit版・64bit版のMSI形式インストーラの提供が始まった。

### Firefox 66

 * 音声の自動再生がブロックされるようになった。
 * タブが多すぎる時に溢れた文が表示されるメニューから、開かれているタブだけを対象に検索できるようになった。
 * アドオンの固有の設定の保存先がJSONからIndexed DBに変更された。
 * アドオンマネージャから各アドオンのショートカットを変更できるようになった。
 * 証明書エラーのページの内容が、よりエラーの詳細が分かるような内容となった。
 * macOSのTouch Barに対応した。
 * コンテントプロセス数を4から8に増やした。
 * Windows 10のWindows Hello（生体認証）に対応した。
