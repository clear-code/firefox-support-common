# README

## Requirements

* TeX
* Pandoc
* Ruby
* Rake
* [Mustache](https://mustache.github.io/)

## Install tools (Debian GNU/Linux)

    sudo apt-get install -y pandoc ruby ruby-bundler rake texlive-luatex texlive-lang-japanese texlive-latex-extra gnumeric

## Install gems

    bundle install

## Generate PDF

    rake
